# ❄️ SnowLuma
> Next Remote Protocol Framework.


## 关于项目
项目处于开发初期，仅支持部分功能，持续高速迭代中，仅支持有头。

## 测试要求
1. 外部库能正常加载

2. 外部库能正常卸载

3. 卸载时Pipe以及副作用被清除

4. 当Snowluma重启时可恢复所有Pipe管道
